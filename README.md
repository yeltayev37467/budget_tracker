## 🚀 How to Use

1. **Download the App**: Install the Budget Tracker App on your device.
2. **Create a Profile**: Add a new profile to start tracking.
3. **Set Up Budgets**: Add categories and assign budgets to each.
4. **Log Expenses**: Enter expenses as they occur and categorize them.
5. **Analyze Spending**: Use the built-in analytics tools to review your financial habits and make improvements.
